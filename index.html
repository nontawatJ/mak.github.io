    Map Tour Planner

    CMPT 353 | SFU | Nov 2020 - Dec 2020

    Tour planning is always a painful process. It wastes lots of time and consumes lots of energy.

    With the opportunity to develop an application that uses big data, we have decided to create a program that will do all the planning for you in the city of Vancouver according to your interest and travel mean.

    Features:

  • Data Cleaning - Implemented a cleaning algorithm that will remove all the unnecessary data from the big data according to the user's interest and travel means.
  • User interest - Implemented an algorithm that uses the user input image to determine the user's point of interest.
  • Find attractions - Used the greedy algorithm to find the path to the nearest five tourists attraction according to the user interest and travel means.
